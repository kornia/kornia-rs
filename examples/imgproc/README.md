Example how to perform basic image processing operations using Kornia Rust library.

```bash
Usage: imgproc --image-path <IMAGE_PATH>

Options:
  -i, --image-path <IMAGE_PATH>
  -h, --help                     Print help
```

Example:

```bash
cargo run -p imgproc -- --image-path path/to/image.jpg
```
