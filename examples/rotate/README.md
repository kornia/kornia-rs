Example showing how to rotate an image using Kornia Rust.

```bash
Usage: rotate --image-path <IMAGE_PATH>

Options:
  -i, --image-path <IMAGE_PATH>
  -h, --help                     Print help
```

Example:

```bash
cargo run -p rotate -- --image-path path/to/image.jpg
```
