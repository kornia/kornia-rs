Example showing how to visualize the results of a model using Kornia and Rerun.io

```bash
Usage: rerun_viz --image-path <IMAGE_PATH>

Options:
  -i, --image-path <IMAGE_PATH>
  -h, --help                     Print help
```

Example:

```bash
cargo run -p rerun_viz -- --image-path path/to/image.jpg
```
