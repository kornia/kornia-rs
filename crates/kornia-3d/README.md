> **3D computer‑vision utilities for Rust.**

## 🚀 Overview
`kornia-3d` provides geometric primitives, point‑cloud processing, and 3‑D transformations built on top of the core Kornia ecosystem.

## 🔑 Key Features
- Point‑cloud data structures
- Camera models and projection utilities
- 3‑D registration and alignment algorithms
- Interoperability with `kornia-tensor` and other crates

## 📦 Installation
```toml
[dependencies]
kornia-3d = "0.1.11"
```

## 🛠️ Usage
```rust
use kornia_3d::pointcloud::PointCloud;
// Example usage of 3D utilities
```
