> **AprilTag detection and decoding for Rust.**

## 🚀 Overview
`kornia-apriltag` provides fast and accurate AprilTag detection, pose estimation, and decoding utilities, integrated with the Kornia tensor and image infrastructure.

## 🔑 Key Features
- Real‑time AprilTag detection
- Support for multiple tag families
- Pose estimation using camera intrinsics
- SIMD‑accelerated processing

## 📦 Installation
```toml
[dependencies]
kornia-apriltag = "0.1.11"
```

## 🛠️ Usage
```rust
use kornia_apriltag::detector::Detector;
// Example code to detect tags in an image
```
