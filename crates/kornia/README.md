> **A low‑level 3D computer‑vision library in Rust.**

## 🚀 Overview
Kornia provides core vision primitives, tensor operations, and utilities for building high‑performance computer‑vision pipelines in Rust.

## 🔑 Key Features
- Strongly‑typed image and tensor structures
- Zero‑copy memory management
- Extensive set of vision algorithms
- Interoperability with Python and C++ bindings

## 📦 Installation
```toml
[dependencies]
kornia = "0.1.11"
```
