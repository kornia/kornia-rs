> **Algebraic utilities for computer vision in Rust.**

## 🚀 Overview
`kornia-algebra` provides linear‑algebra primitives, matrix operations, and geometric transformations that underpin many vision algorithms in the Kornia ecosystem.

## 🔑 Key Features
- Generic vector and matrix types
- Efficient SIMD‑accelerated operations
- Compatibility with `kornia-tensor`
- Support for homogeneous coordinates and transformations

## 📦 Installation
```toml
[dependencies]
kornia-algebra = "0.1.11"
```

## 🛠️ Usage
```rust
use kornia_algebra::matrix::Matrix4;
// Example: create a transformation matrix
```
