# kornia-algebra

A core library for geometric algebra in the [kornia-rs](https://github.com/kornia/kornia-rs) ecosystem, providing types for linear algebra and Lie theory designed for computer vision and robotics.

Built on top of [glam](https://github.com/bitshifter/glam-rs) with SIMD acceleration.

## Documentation

See the [crate documentation](https://docs.rs/kornia-algebra) for usage examples and API reference.
